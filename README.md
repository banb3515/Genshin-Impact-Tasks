> ### **Genshin Impact Tasks란?**
>
> -   Genshin Impact Tasks는 게임 [원신](https://genshin.mihoyo.com/ko)의 도우미 앱 입니다.
> -   할 일을 기록해두거나 게임 플레이에 도움이 되는 기능을 사용할 수 있습니다.
> -   할 일 목록은 Firebase - Realtime Database에 저장되며, Android와 Windows와 연동이 가능합니다.
